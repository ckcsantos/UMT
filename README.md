# UMT

UMT is a new project scaffolded as a Python package. It includes a simple command-line entry point, basic package structure, and unit tests.

## Getting Started

1. Create a virtual environment:
   ```bash
   python3 -m venv .venv
   source .venv/bin/activate
   ```
2. Install dependencies:
   ```bash
   pip install -U pip
   pip install -r requirements.txt
   ```
3. Run the sample script:
   ```bash
   python -m umt
   ```

## Project Structure

- `src/umt.py` - main application module
- `tests/test_umt.py` - example unit test
- `pyproject.toml` - project metadata
